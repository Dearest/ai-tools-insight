### 1. 产品介绍和使用方式  
**Monica** 是一款基于 ChatGPT API 的 Chrome 扩展，作为个人 AI 助手，提供聊天、文案创作、文本解释/翻译/改写等功能。  
**使用方式**：  
- 在任意网页与 Monica 聊天  
- 使用 80+ 模板生成文案并插入网页  
- 选中文本后让 Monica 解释、翻译或改写
